# 실험 계획

## 고정할 것

- 동일한 `landmark_split.json`
- 동일한 검증 triplet seed와 `hard-val-fraction=0.5`
- 동일한 epoch, batch 구성, learning rate
- 한국 데이터가 없음을 증명하는 동일한 `data_audit.json`
- 각 조건을 seed 42, 43, 44로 세 번 실행

## 한 번에 하나만 바꾸는 순서

| 실험 | 이전 실험에서 바꾸는 한 가지 | 확인 목적 |
|---|---|---|
| E0 | CE만 사용 | 가장 단순한 기준선 |
| E1 | batch-hard triplet loss 추가 | 실제 순위 조건 개선 여부 |
| E2 | average pooling을 GeM으로 변경 | 건물의 강한 지역 특징 보존 여부 |
| E3 | 512차원 projection 추가 | 압축 임베딩 효과 |
| E4 | center crop을 pad로 변경 | 가장자리 랜드마크 보존 효과 |
| E5 | random 50% + hard 50% 검증으로 선택 | 쉬운 검증 점수 착시 제거 |
| E6 | 비한국 국가 단위 검증 split 사용 | 처음 보는 국가로의 전이 확인 |
| E7 | 고정 학습 split에서 hard-negative 파인튜닝 | 닮은 다른 랜드마크 구분 효과 |
| E8 | flip TTA 추가 | 추론 안정성 효과 |

## 매 실험에 저장할 숫자

- 세 seed 각각의 `val_triplet_accuracy`
- 세 seed 평균과 표준편차
- `mean_sim_ap`, `mean_sim_an`, `mean_margin`
- 학습 시간, 추론 이미지/초, 최대 GPU 메모리
- 체크포인트 SHA-256
- 사용한 명령 전체

Accuracy가 같으면 `mean_margin`이 큰 모델을 선택합니다. 모든 설정 선택이 끝난 뒤에만
숨겨진 최종 테스트 결과를 한 번 확인합니다.
